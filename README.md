# AURA-RC-PLANE-

<img width="742" height="680" alt="Screenshot 2026-07-08 161159" src="https://github.com/user-attachments/assets/0354225e-a7c3-4fc1-893a-fcf8836f581f" />

I made a custom Tailless twin motor RC Plane with custom Flight Controller (based on STM32). The Flight Controller  will have a  IMU's and a barometer for precise data collection. The Plane design will include a tailless structure and the motions will be controlled by the flaps on the wings.

## Features
- Base on  STM32F401CBUx 
- Has dual IMU (MPU_6050)
- Has a Barometer (BMP280)
- Uses 2 Light-Weight Servoes
- Uses Custom flight-controller
- Tailless twin motor Design
